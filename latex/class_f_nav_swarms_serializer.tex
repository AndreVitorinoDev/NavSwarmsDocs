\doxysubsection{Detailed Description}
Saves the FNav\+Voxels grid array into a .vox binary file with the given name so it can be later read, the file is always stored in the "{}/\+Content"{} folder, and it\textquotesingle{}s path is given to the \doxylink{class_u_nav_swarms_data_asset}{UNav\+Swarms\+Data\+Asset}, if it is deleted or modified the created file will also be deleted or modified.
