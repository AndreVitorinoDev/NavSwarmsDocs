\doxysubsection{Detailed Description}
Threaded processing of the pathfinding, it has 2 modes\+:
\begin{DoxyItemize}
\item Persistent\+: by setting persistent to true the pathfinding will continuously run until stopped.
\item Non\+Persistent\+: by setting persistent to false, the pathfinding will only run once.
\end{DoxyItemize}

\begin{DoxyRemark}{Remarks}
Independent of mode of operation, when a path is found it will update the given \doxylink{struct_f_path_parameters}{FPath\+Parameters} path 
\end{DoxyRemark}
