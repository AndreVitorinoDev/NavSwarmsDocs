\doxylink{class_nav_swarms}{Nav\+Swarms} is a Unreal plugin to provide 3D pathfinding for Unreal Engine 5. Primarily designed for baking large voxel grids, and using the provided \doxylink{class_a_nav_swarms_manager}{ANav\+Swarms\+Manager} to queue pathfinding queries using an optimized custom A\texorpdfstring{$\ast$}{*} algorithm.

This plugin is focused on providing an easy and efficient way to manage flying AI which needs a system that is not covered by the Unreal native AI navigation system.

Key features\+:
\begin{DoxyItemize}
\item Navigation Manager Actor for configuring the map navigation grid
\item API to manage pathfinding queries and retrieve finished paths
\item Threaded pathfinding which can run continuously or just once
\item Baked pathfinding grid, allowing fast load on large maps
\item Efficient debug visualization of grid size, occupied voxels and even current paths
