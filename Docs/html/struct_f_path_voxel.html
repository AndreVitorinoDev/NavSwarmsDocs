Detailed Description

Intermediary struct to hold the necessary variables for the A* algorithm to function, necessary due to running in threads and to avoid modifying the grid FNavVoxels directly. One array with the size of the grid is created per pathfinding query, as such, running threads in persistent mode is better for memory since it only has to create this array once and re-use it in the subsequent runs.
