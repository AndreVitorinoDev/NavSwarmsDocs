\doxysubsection{Detailed Description}
Responsible for creating and saving the current navigation grid, keeping important information about the grid. Since the grid is stored on a binary file (.vox) located in the root "{}content"{} folder, it keeps track of what file is associated with it, if the UData\+Asset is deleted or modified, the "{}.\+vox"{} file will also be deleted/modified Also contains some useful methods to convert from world locations to linear array indexes and vice versa
