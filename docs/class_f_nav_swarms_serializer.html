Detailed Description

Saves the grid data into a .vox file.

Saves the FNavVoxels grid array into a .vox binary file with the given name so it can be later read, the file is always stored in the "/Content" folder, and it's path is given to the UNavSwarmsDataAsset, if it is deleted or modified the created file will also be deleted or modified.
