Detailed Description

Individual Voxel in a grid.

Holds only it's most important values such has position, self index in the grid array and if it's occupied or free. Should not be modified after creation or loading.

Remarks
Neighbours array doesn't have a fixed size due to voxels on sides and corners of the grid, however, other voxels are expected to have 26 unique neighbours (All adjacent voxels including diagonal ones).
