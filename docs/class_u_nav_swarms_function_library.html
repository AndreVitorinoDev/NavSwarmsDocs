Detailed Description

Blueprint helper class.

Directly communicates with a ANavSwarmsManager in order to manage the pathfinding queries or to get points/ directions on a path, it can also be used directly via code.

Remarks
If there's more than one ANavSwarmsManager it chooses the first one it finds, unless a specific one is passed down. The ID for a query using this class will be the Actor's GetUniqueID().
