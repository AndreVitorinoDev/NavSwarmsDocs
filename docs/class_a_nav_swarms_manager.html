Detailed Description

Entry point for all navigation logic.

This class is responsible for holding the information about the current grid and managing pathfinding queries, a scene that requires pathfinding has to have one with an associated FNavSwarmsDataAsset

Remarks
should be possible to use more than one per scene, however, UNavSwarmsFunctionLibrary will only work for the first one it finds unless a specific ANavSwarmsManager is given.
