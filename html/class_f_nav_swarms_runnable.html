Detailed Description

Threaded processing of the pathfinding, it has 2 modes:

  • Persistent: by setting persistent to true the pathfinding will continuously run until stopped.
  • NonPersistent: by setting persistent to false, the pathfinding will only run once.
Remarks
Independent of mode of operation, when a path is found it will update the given FPathParameters path
